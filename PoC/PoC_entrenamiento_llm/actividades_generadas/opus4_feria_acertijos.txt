LA FERIA MATEMÁTICA DE LOS NÚMEROS MÁGICOS
OBJETIVO DE APRENDIZAJE
Objetivos principales de matemáticas:

Consolidar operaciones básicas (suma, resta, multiplicación) adaptadas al nivel de cada estudiante
Introducir/reforzar el concepto de fracciones mediante representación visual y manipulativa
Desarrollar el cálculo mental y la estimación

Objetivos curriculares transversales:

Lengua: Expresión oral al explicar procesos matemáticos y crear instrucciones
Ciencias Naturales: Aplicar el método científico al comprobar hipótesis numéricas
Educación Artística: Diseño y creación de materiales visuales
Competencias sociales: Colaboración, respeto de turnos y comunicación asertiva

DESCRIPCIÓN DE LA ACTIVIDAD
Los estudiantes crearán y gestionarán una "Feria Matemática" con diferentes puestos/estaciones. Cada puesto representa un desafío matemático diseñado por y para los propios estudiantes, adaptado a diferentes niveles de dificultad. La actividad combina la creación de materiales, la resolución de problemas y la enseñanza entre pares.
La feria tendrá 4 estaciones principales que rotarán, permitiendo que todos experimenten diferentes roles: creador, explicador, participante y evaluador. Cada estación abordará un concepto matemático desde diferentes perspectivas sensoriales y niveles de complejidad.
ROL DEL PROFESOR
Fase de preparación (30-40 minutos):

Presentar el concepto usando una demostración visual de una mini-estación ejemplo
Facilitar la formación de parejas estratégicas (se detallan más adelante)
Circular entre grupos ofreciendo andamiaje diferenciado:

Para Elena: Proporcionar estructura visual clara del proceso
Para Luis: Ofrecer breaks de movimiento cada 10 minutos
Para Ana: Plantear extensiones desafiantes
Para María y Sara: Verificar comprensión mediante preguntas orales



Fase de ejecución (45-60 minutos):

Gestionar las rotaciones cada 15 minutos aproximadamente
Documentar el progreso individual sin interrumpir
Intervenir solo cuando sea necesario para:

Reconducir la atención (especialmente Luis)
Validar estrategias alternativas (Ana, Emma)
Reforzar positivamente los logros (todos, especialmente Elena)


Tomar nota del tiempo real que cada pareja necesita, permitiendo flexibilidad

Fase de cierre (15-20 minutos):

Moderar la reflexión grupal
Destacar diferentes estrategias utilizadas
Conectar con futuros aprendizajes

ADAPTACIONES METODOLÓGICAS DUA
Para Elena (TEA nivel 1):

Representación: Tarjetas visuales con el proceso completo visible desde el inicio. Código de colores para cada tipo de operación
Acción/Expresión: Tiempo extra sin presión. Opción de comunicar comprensión mediante fichas de colores en lugar de verbal
Implicación: Rutina clara: siempre empieza en la misma estación, mismo orden de rotación

Para Luis (TDAH):

Representación: Instrucciones en trozos de 2-3 pasos máximo. Timer visual para gestionar tiempo
Acción/Expresión: Rol activo de "mensajero matemático" entre estaciones. Puede resolver de pie o moviéndose
Implicación: Recompensas inmediatas (pegatinas) por cada mini-logro. Breaks programados

Para Ana (Altas capacidades):

Representación: Acceso a conceptos avanzados (porcentajes, decimales) mientras otros trabajan fracciones
Acción/Expresión: Libertad para crear métodos alternativos de resolución y documentarlos
Implicación: Rol de "consultora experta" que otros pueden solicitar (fomenta paciencia)

Para María y Sara (apoyo medio):

Representación: Grabaciones de audio con las instrucciones que pueden reproducir
Acción/Expresión: Permitir explicaciones orales de sus procesos en lugar de escritos
Implicación: Roles que aprovechan sus fortalezas comunicativas

Para resto del grupo:

Representación: Múltiples formatos disponibles (visual, auditivo, manipulativo)
Acción/Expresión: Elección del método de resolución y presentación
Implicación: Conexión con intereses personales en los contextos de problemas

INSTRUCCIONES PASO A PASO
FASE 1: Introducción y Modelado (10 min)

Profesor muestra mini-estación ejemplo: "El Puesto de Fracciones de Pizza"
Demostración visual de cómo crear materiales y explicar a otros
Presentación de las 4 estaciones base que crearán

FASE 2: Creación en Parejas (30 min)

Cada pareja recibe su "kit de creador" con materiales base
Primeros 10 min: Lluvia de ideas y bocetos

Elena y Hugo: Crear esquema visual primero
Luis: Puede caminar mientras piensa


Siguientes 15 min: Construcción de materiales

Ana: Crea versión avanzada y versión base
María/Sara: Graban instrucciones de audio


Últimos 5 min: Prueba rápida entre la pareja

FASE 3: La Feria en Acción (45 min)

Rotación 1 (15 min): Parejas en estaciones asignadas
Rotación 2 (15 min): Cambio diagonal
Rotación 3 (15 min): Nuevo cambio
Entre rotaciones: Luis lidera "estiramiento matemático" de 1 minuto

FASE 4: Reflexión y Celebración (15 min)

Círculo de compartir (respetando quien prefiere no hablar)
Cada pareja muestra su descubrimiento favorito
Creación conjunta del "Libro de Trucos Matemáticos de 4º"

Fichas/tapones reciclados para contar

Rotuladores, tijeras, pegamento
Cajas de zapatos para crear las "estaciones"

Plantillas de secuencias para completar
Fichas de colores para patrones
Carpeta organizadora con fundas transparentes

María + Sara:

Grabadora o tablet para audio
Tarjetas de historias ilustradas
Micrófono de juguete para presentaciones

Alex + Emma:

Lupas para "investigar" números
Piedras/minerales para clasificar y contar

Materiales creados durante la actividad:

"Pasaporte matemático" individual para sellar logros

ESTRATEGIAS DE MOTIVACIÓN
Conexiones con intereses personales:
Para amantes de las ciencias (Alex, Hugo, Ana):

Problemas contextualizados en experimentos: "Si un científico mezcla 2/3 de solución A..."
Datos curiosos: "¿Sabías que el corazón late 100,000 veces al día? ¿Cuántas en una hora?"

Para Elena (patrones y orden):

Descubrir la "secuencia secreta" en cada estación
Premio especial por la estación más organizada

Para Luis (movimiento):

Incorporar saltos para representar multiplicaciones
"Carreras de relevos matemáticos" entre estaciones

Para Emma (geología):

Problemas sobre clasificación de minerales por peso/tamaño
"¿Cuántas gemas de cada tipo necesitas para...?"

Para María y Sara (historias y teatro):

Crear personajes matemáticos con voces diferentes
Mini-obra sobre "El día que los números se escaparon"

Elementos sorpresa para mantener curiosidad:

Sobre misterioso con "Mega-reto grupal" para el final
Datos asombrosos relacionados: "Un billón de segundos = 32,000 años"
Conexión con la vida real: "Matemáticas de supervivencia" para camping

Sistema de reconocimiento inclusivo:

Medallas por diferentes logros: creatividad, colaboración, perseverancia, precisión
Cada estudiante elige qué logro destacar de sí mismo
Mural colectivo donde cada uno aporta su "truco matemático favorito"
